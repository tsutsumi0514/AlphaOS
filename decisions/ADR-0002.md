# ADR-0002: Use Python + FastAPI + SQLite for the First MVP

## Status
Accepted

## Context
AlphaOS needs a simple and maintainable foundation for the first MVP. The project should start small, support quick iteration, and avoid unnecessary complexity.

## Decision
We will use:
- Python for the backend
- FastAPI for the web API
- SQLite for local storage in the MVP
- A simple LINE-first or minimal web interface for output

## Consequences
- Development will be fast and easy to change
- The MVP can stay small and understandable
- We can later upgrade storage or UI if needed
- We avoid overengineering at the start
